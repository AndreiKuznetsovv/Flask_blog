## General info

This project is simple full-featured web-application
created with application factories pattern. <br>
Demo:
![Blog screenshot](./flask_github.png?raw=true "Blog demo")
## Functionality

* Basic user authentication system (with email and password)
* Profile - possibility to change Email/username/profile picture.
* Password change via email
* CRUD (Create Read Update Delete) posts
* Likes
* Comments

## Technologies

* Flask
* SQLAlchemy
* Flask-Login
* Flask-WTF
* Flask-Migrate
* Bootstrap 4.0 for css and js
* Blueprints with Application Factories pattern
